##### 列出远程分支

```sh
git branch -r
```

##### 列出远程和本地分支

```sh
git branch -a
```

##### 分支增删改

```sh
# 增
git branch branch_name [start_point]
# 删
git branch -d|-D branch_name
# 改
git branch -m|-M branch_name new_name
```

##### git-rebase

git rebase用于把一个分支的修改合并到当前分支
